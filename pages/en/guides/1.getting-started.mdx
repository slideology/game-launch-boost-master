# 快速开始

## 环境准备

本项目使用 pnpm 作为包管理器，请先安装 pnpm：

```bash
npm install -g pnpm
```

## 安装步骤

1. 从哥飞社群对应帖子获取到项目源码后解压，使用 IDE 打开项目

然后用终端执行以下命令：

2. 安装依赖
```bash
pnpm install
```

3. 本地运行项目
```bash
pnpm dev
```

## 部署说明

请将你的代码先上传到 GitHub 上，然后按照以下步骤进行部署：

[部署到 Cloudflare Pages](/en/guides/2.depoly-2-cloudflare-pages) - 在 Cloudflare Pages 上部署项目
